Final project: Configure SMTP server on an EC2 instance

Our goal was to create a mail server that could handle at least sending mail for multiple users

Services used:

  • Postfix (Mail Transfer Agent)
  • Snail (Mail Client)

    • Challenges:

  • Email being marked as spam
  • Configuring functionality for reciving emails

    • Sample usages:

  • echo 'This is a test email from Team Garbage-Bin!' | s-nail -s 'Hello Steve!' stevev@hexadecimal.uoregon.edu
  • cat draft.txt | s-nail -s 'Hello Steve!' stevev@hexadecimal.uoregon.edu
